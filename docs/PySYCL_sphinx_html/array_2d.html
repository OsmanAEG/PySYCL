PySYCL: Array2D Sub-Module#

Array2D module for PySYCL

This module provides classes and functions for creating PySYCL 2D arrays.

class pysycl.array_2d.array_2d_init(self: pysycl.array_2d.array_2d_init, rows: int, cols: int, device: pysycl::Device_Instance)#

Bases: pybind11_object

Description

This class creates a PySYCL array_2d object.

Default Constructor

Constructor that creates a 2D PySYCL array.

Parameters
cols: int

Number of columns.

rows: int

Number of rows.

device:

Optional: The target PySYCL device.

Returns

A PySYCL 2D array.

Example
>>> import pysycl
>>> A = pysycl.array_2d.array_2d_init(10, 12)
