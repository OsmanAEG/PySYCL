vector_operations module

Vector operations sub-module for PySYCL

This sub-module contains functions that perform simple vector operations and returns the results.

vector_operations.vector_addition(vector_a: List[float], vector_b: List[float], platform_index: int = 0, device_index: int = 0) → List[float]
Description

Performs a vector addition on two python lists and returns the sum.

Parameters
vector_alist[float]

The first vector to be added.

vector_blist[float]

The second vector to be added.

platform_indexint, default = 0

Optional: The index of the device platform to be used.

device_indexint, default = 0

Optional: The index of the device to be used for a given platform.

Returns
list[float]

The sum of the two input vectors.

Examples
>>> from pysycl import vector_operations as vec_ops
>>> vector_a = [1.0, 2.0, 3.0]
>>> vector_b = [4.0, 5.0, 6.0]
>>> vec_ops.vector_addition(a, b)
[5.0, 7.0, 9.0]
